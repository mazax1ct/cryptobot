Торговый бот — это программа, используемая для автоматизации сделок и принятия решения о покупке или продаже на основе рыночных данных и заранее определенных правил.

Для начало работы Вам нужно связать свою биржу по API ключам с торговым ботом и установить определенные параметры, чтобы указать боту, как действовать.
